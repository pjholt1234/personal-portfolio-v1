Twin Cities Project

Date: 19/10/2020 - 08/03/21

Description: The twin cities project was my first real step into integrating many different technologies for the purpose of one website. This was a large group project spanning half of my second year at university. This consisted of creating a MySQL database as the backbone of the website and connecting weather, social media and map API’s together for the final result.

From this I learnt, how to create a full stack web app with a MySQL and php backend, and JavaScript and html frontend. I learnt a number of skills including structure my website in an object-oriented way allowing the website to shape itself around the content selected by the user, as apposed to hardcoding everything. Myself and my group didn’t spend much time on the presentation of the website as it wasn’t marked, I think this is reflected as the appearance is lacking.

The website allows the user is able to see various tourist attractions and notable landmarks in Liverpool and shanghai (the twin cities we chose to cover). User can mouse over the icons on the map and information about the landmark is displayed in a pop up. If the user clicks the icon, they are taken to an info page, where images are pulled from the server and Flickr. The Demo linked does not contain the live weather updates due to https restrictions on my website.

Tool's / API's / Etc Used

  • Openweathermap API
  • Google Maps API
  • Twitter API
  • Flickr API
  • mySQL Database
  • jQuery
